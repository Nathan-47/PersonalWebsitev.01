Static One Page Filter Portfolio

This project is a portfolio that is currently a placeholder and can be used for anyone that needs a portfolio for their work. The portfolio has a filter option that can be altered to the area or job title of choice for example on this placeholder the three filter options are; graphic design, front-end development and 3D Modelling. The project is responsive on all devices


At first the project was created for myself as a one-page portfolio to show my work, that I completed at university. The filter options as stated before were catered to me as I had work from university that can be placed in all three of the categories. Each piece of work within the portfolio also has an option to be viewed which shows all details of the work. For the front-end development section of work, I added two buttons which are Live Demo and Code. Live Demo gives user the ability to look at the live site via GitHub or other domain hosts. The code button when the user clicks it show the files of the website along with the readme doc all via GitHub.

The coding editor tool used for this work is Brackets. The software that I used for this work is HTML5, CSS3 and JavaScript. The portfolio also uses a smooth scroll that allows a smooth user experience. I also made the navigation as simple as possible with the nav-links at the top being; About, Projects, Skills and Contact. I added a stickynav script so that users can instantly navigate to another section of the page in an instant without scrolling all the time.


The portfolio is clear and concise allowing an easy experience for the user. When creating the site, I wanted to be able to show the work clearly and smoothly using modals. I wanted to show the work in the modal as I wanted to display my progression of the work via images and videos and this was done by implementing a carousel within the modal. As it’s a portfolio I knew that the colours have to complement each other so that the user can have an easier reading experience. The colours that I used are Blue, white and black with black being the font on the majority of the portfolio. The final little amendment I made was adding a mini summary of each project/work such as the software/hardware tools used i.e. – Adobe Photoshop and a tiny statement of what the project is.
