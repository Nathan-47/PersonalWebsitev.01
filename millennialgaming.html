Millennial Gaming (PHP)

This project was made at my time at university. The aim of this piece of work was to design and create an interactive website/application for a business to show a section on a site to showcase particular products. The big note is that the website is not be a complete website. The site I created was a two page gaming ecommerce site that will show a variety of games to choose from at good prices.


The project was created for a module and the mission was to create an interactive website/application for a business to show the products and further details on said product. Creating this project allowed me to show my skills with PHP and using and understanding databases.

HTML5, CSS3, JavaScript and PHP are used to create this piece of work whilst using the program xampp that runs a localhost database allowing me to integrate PHP into my work and collect data from the database which is then relayed to the site.


The website I was told didn’t need to have a homepage and just had to show the product page and the information of the product located on an external page. I knew that I had to create two pages one for the products to be displayed on. On this page I chose to show 12 games in a grid layout for customers to see clearly. I purposely made the thumbnail of the games bigger than the game titles and prices. The intention was to hope that the thumbnail would catch the attention of the customer straight away.

I called the second page ‘overview.php’ as this will be the page that will show all the information for each of the games. How I show games individually on this page is by using a php if statement - if(isset($_GET['ID'])). Using this made it much easier to display each of the games on one page as the sql would select data from the database containing all data for this work which will then read the gameID number and display all information contained within this row. For example, DOOM is the 8th game on the grid therefore the gameID is 8, when the user clicks this game it will direct them to the overview page displaying all DOOM’s details which is all stored in the row of gameID 8 within the selected database.
