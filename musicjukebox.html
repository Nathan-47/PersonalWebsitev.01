Music Jukebox (PHP)

This project is an application for online music similar to a music jukebox. Again, this project was created at my time in university and is only viewable on desktop. Users can use this work to play a selection of music from a small number of artists available.


The mission statement for this project was to create a music application containing songs from various artists of your choice and allow their music to be played via media player. In this project I used the same PHP method as the Millennial Gaming project to make sure the player on the separate overview page plays the correct song when the user clicks on an artist.

The application includes the following; HTML5, CSS3, PHP and MySQL to form a website viewable on a desktop. Our module leader allowed the use of W3Schools templates to help aid the structure of the website.


I knew that I had to set a narrative for the site, I did this via naming the site URBAN Trackz and made the landing page heading an image of famous rap artists. This will connote to the user that the site will include rap content . Using inspirations such as the BBC radio desktop application and MTV music charts player helped me understand what all users want to know. I included the main elements such as; the artists name, song and the album this can be seen in photo (middle-right). The usability is simple, once the users click on the song of their choice, the user will land on the overview page that will display the picture of the album cover, artist and album title. The overview page will contain a music player that allows the user to play, pause, scrub through and change the volume of the song.
