Birmingham Impact Football Club

The project is a fully working and responsive football website made for the Birmingham Impact Football Club that play in both U11’s and U13’s Central Warwickshire league. The project includes information about the club varying from; teams, contact details, registration forms, club legacy and a landing page giving information on news and why players should join the club.


As stated, before the main focus was to create a site that is usable and interactive on all devices as the old site was not. Nowadays mobile phones are being used more and more as they are more accessible thus the focus of attention being on the responsiveness of mobile /tablet devices and user experience.

The technology used for this project was brackets web editor which is my main web editor tool when I created and designed the site. The software to create this site was HTML5, CSS3, JavaScript and use of a third party form company to help with both forms created on the site.


The site has a sticky navbar to make navigation much easier for the user instead of having a back top button. The form is a major staple within the site as users will need to send all football related information to the club’s head coach. I knew that I would need to have a form that would include a multiple choice dropdown selector whilst being able to be sent information on the form using a localhost server. Knowing this I decided to search for any third party PHP form companies that would suit what I was looking for. I stumbled upon a great form provider such as formspree that allow users to use their database to safely store and send information to the club email.
