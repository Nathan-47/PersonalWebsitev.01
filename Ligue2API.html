Ligue 2 API

This project was made to showcase the use of API on a website and using standard HTML and CSS and minimal JS to compliment the API data. The site allows football supporters from any league or most specifically Ligue 2 to view the latest news from last season 2019/20 to the new season 2020/21.


I created this project to show that I can create a site using API. The data extracted was from a site called APIFootball.com. When progressing through this project I began to understand more about API and how it works. Making this project will let me implement more API into future work.

The project created with JavaScript, HTML5, CSS3, API, PHP and Web editor used was Brackets


This website doesn’t have much animation or user interactivity as I just wanted to focus on the API side and less on the user experience. The layout of the site is clear and well-spaced, I took inspiration from other football sites such as Bundesliga and EFL Championship. The project isn’t a full football league site like the sites stated before, when creating the site, I checked on the APIFootball.com site and it only returned data on certain parameters. With that I decided to go with the fixture & results, standings and match result parameters. This way I can work small and get more of an understanding of how to work with JSON strings and parameters. As seen in the pictures the site follows a certain colour scheme that is blue, white and light green the same as the Ligue BKT colours theme. The site isn’t responsive on all devices and is only created for desktop viewing.
